# DWS Blog

## Stack

- TypeScript
- React 19
- Sass (SCSS)

## Running

Make sure you have the following environment variables:

| Variable name         | Expected value           |
| --------------------- | ------------------------ |
| `PUBLIC_API_BASE_URL` | The public API base URL. |

### Development

```shell
npm run dev
```

### Production

Run the following command to generate the production bundle:

```
npm build
```

Configure your deployment to serve the contents of the `/dist` folder in order to run the production build.
